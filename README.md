## Features (Only screens):

- Home Page
- Product Listing Page
- Filters by:
  - Sort
  - Price
  - Brand
  - Ratings
- Cart Management
- Wishlist Management
- Authentication:
  - Sign-up
  - Login
  - Logout
- Search
- Address

## Tech Stack:

- HTML
- CSS

## Run locally:

Clone the project

```
  git clone https://github.com/rohanmathur91/evolt-shop.git
```

Go to the project directory

```
  cd evolt-shop
```

```
  open the index.html in chrome
```
